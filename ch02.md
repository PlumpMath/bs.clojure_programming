ch02. Functional Programming
==========================================


```clojure
(#(map * %1 %2 %3) [1 2 3] [4 5 6] [7 8 9]) ;=> (28 80 162)
(#(apply map * %&) [1 2 3] [4 5 6] [7 8 9]) ;=> (28 80 162)
((partial map *)   [1 2 3] [4 5 6] [7 8 9]) ;=> (28 80 162)

((comp str +) 1 2 3 4) ;=> "10"

(-> 3
    (/ 2)) ;=> 3/2

(->> 3
     (/ 2)) ;=> 2/3
```
